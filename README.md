# Firat Kara website

Static website for `firatkara.nl`, hosted on GitHub Pages.

## Editing

Use this GitHub repository as the only source for website edits. Do not edit
the public website through WordPress; update files here, then commit and push
to `main`.

Most content is in `index.html`.

Common Codex requests:

- "Add this publication to the selected publications list."
- "Update the About paragraph with this text."
- "Replace the CV file with this new document."

After editing, commit and push to `main`. GitHub Pages publishes the site automatically.

## Domain

The repository contains a `CNAME` file for `firatkara.nl`.
